# How to test

## Overview

A **Universally Unique Identifier (UUID)**, also known as a **Globally Unique Identifier (GUID)**, is a 128-bit value.  
Systems use UUID to uniquely identify information across distributed environments.

UUID remove the need for a central authority to assign identifiers.  
Use UUID for resources, database records, or system components.

This article provides example UUID for testing, development, and placeholder use.

## Resolution

If you need **Version 4 (random) UUID** immediately, use the following examples.  
These values are valid and suitable for non-production scenarios.

- `4a1f****-****-****-4f2e`
- `f9d7****-****-****-7c3b`
- `e2a5****-****-****-3e7f`
- `b6c4****-****-****-1b3e`
- `d3f1****-****-****-5b2d`
